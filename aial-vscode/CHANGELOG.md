# Changelog

## 0.1.0 — 2026-05-11
- Initial release
- Syntax highlighting: keywords, types, strings, numbers, operators
- Function and method call highlighting
- Path expression coloring (`context::new`, `file::read`)
- Built-in function recognition (`println`, `strlen`, `strslice`, etc.)
- Comment toggling (`//`, `/* */`)
- Bracket matching and auto-closing
